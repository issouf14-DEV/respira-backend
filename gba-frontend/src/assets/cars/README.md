# Images de voitures pour LE-GBA

Ce dossier contient les images de voitures pour le site.

## Instructions pour ajouter des images

1. Téléchargez des images de voitures depuis :
   - https://unsplash.com/s/photos/luxury-car
   - https://unsplash.com/s/photos/toyota
   - https://unsplash.com/s/photos/mercedes-benz
   - https://unsplash.com/s/photos/bmw
   - https://unsplash.com/s/photos/audi

2. Nommez les fichiers selon ce format :
   - `car-1.jpg`
   - `car-2.jpg`
   - `car-3.jpg`
   - `car-4.jpg`
   - etc.

3. Ou utilisez ces URLs gratuites depuis Unsplash :
